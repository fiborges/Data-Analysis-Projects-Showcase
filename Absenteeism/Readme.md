# Absenteeism Data Cleaning and Exploration

## Overview

The Absenteeism Data Cleaning and Exploration project focuses on preparing and cleaning the dataset related to employee absenteeism. Beyond data cleaning, we aim to explore the cleaned dataset to gain insights into absenteeism patterns and potential influencing factors.

## Objectives

1. **Data Cleaning:**
   - Address missing values, outliers, and inconsistencies in the absenteeism dataset.

2. **Exploratory Data Analysis (EDA):**
   - Conduct exploratory analysis to uncover patterns and trends in the cleaned data.

3. **Data Documentation:**
   - Provide comprehensive documentation on data cleaning procedures and exploratory findings.

## Key Activities

- **Missing Value Imputation:**
  Implement strategies to handle missing values in the absenteeism dataset.

- **Outlier Detection and Handling:**
  Identify and address outliers that may impact the accuracy of the analysis.

- **Exploratory Data Analysis:**
  Explore key features, correlations, and trends in the cleaned dataset.
